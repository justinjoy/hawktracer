Calls a calloc() function, without calling a calloc() hook.

The function should be used for allocating memory in an allocation hook to avoid infinite recursion.

Parameters
numnumber of elements to allocate.
sizesize of each element.
Returns
on success, a pointer to the memory block allocated by function. On failure, NULL value.
