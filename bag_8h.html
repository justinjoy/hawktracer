HT_API HT_ErrorCode ht_bag_add ( HT_Bag *  bag,
void *  data 
)

Adds element to the end of the container.

Parameters
baga pointer to the container.
dataa value to be added to the container.
Returns
HT_ERR_OK, if element was successfully added to the container; otherwise, appropriate error code.

The function might fail, if the process can't allocate enough memory to extend internal data buffer (HT_ERR_OUT_OF_MEMORY). The function never fails, if ht_bag_init() doesn't fail and number of elements in the bag is less than min_capacity.

◆ ht_bag_deinit()

HT_API void ht_bag_deinit ( HT_Bag *  bag)

Uninitializes bag structure.

This function releases all the memory allocated for the bag and resets all the fields. After this function call, bag should no longer be used (unless ht_bag_init() is called on the same object).

Parameters
baga pointer to the container.
